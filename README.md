This project analyzes Russian loanwords in Georgian.

Files included:
- dataset.py → script that extracts Russian loanwords in Georgian from Wiktionary
- geo_borrowed_from_russian.json → raw dataset saved as a JSON file
- russianloans_dictionary_structure.py → script that reads the raw dataset and creates a new dictionary with the selected structure
- dictionary_categories.json → prototype dictionary generated from the raw dataset
- russianloans.py → manually completed structured dataset focusing on the first 100 words
- data_analysis.py → script used to count category frequencies and percentages
- full_report.docx → written analysis of the project

How results were obtained:
- Words were manually translated and categorized
- Python was used only to count frequencies and percentages
